## notes

* webcli is a wallet too, not just a client
* “dapp support” means it can connect to or interact with apps/contracts, not just send transactions
* “unclear” means dapp support is not clearly verified
* third-party does not mean bad, just not official

## safety

this list is not an endorsement

* don’t paste seed phrases or private keys into random sites
* check urls twice
* prefer official tools when possible
* test third-party wallets with a fresh wallet first
* assume clones and fake links will exist
